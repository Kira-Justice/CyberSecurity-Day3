# 🐧 Linux Basics

## 🔹 Users
A user is an account that can access the system with specific permissions.

## 🔹 Groups
A group is a collection of users with shared permissions.

## 🔹 File Ownership
Each file has:
- Owner (user)
- Group

---

## 🔹 Commands
- whoami → shows current user
- id → shows user ID and groups
- ls -l → shows file permissions
